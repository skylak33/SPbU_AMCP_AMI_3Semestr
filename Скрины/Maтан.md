# Линейное пространство
Множество L называется #линейным_пространством над $\mathbb{R}$ или вещественным линейным пространством, если выполняются следующие условия для
$\forall a, b,c \in L, \;\; \forall \alpha, \beta \in \mathbb{R}$
1) a + b = b + a
2) (a + b) + c = a + (b+ c)
3) существует нулевой элемент
4) существует обратный элемент
5) $\alpha(\beta a)$ = ($\alpha \beta$)a
6) ($\alpha +\beta) a$ = $\alpha a +\beta$a
7) $\alpha$(a+b) = $\alpha a + \alpha b$
8) 1a = a

# Нормированное пространство. Норма
#Нормированным_пространством Х называется линейное пространство,  в котором введено отображение ||.||: X -> R, называемое #нормой и выполняется ряд свойств: 
$$1) \;\|x\| >= 0, \|x\| = 0 \leftrightarrow x = \mathbb{O} $$
$$2) \; \|\lambda x\| = |\lambda| \|x\| \;\text{(однородность)}$$$$3)\; \|x + y\| \le\ \|x\| +\|y\|\;\text{Неравенство треугольника}$$ Важнейшие примеры нормированных пространств:
1. Конечномерные евклидовы пространства  ;
2. Пространство непрерывных на отрезке функций C\[a, b].\

# Предел функции многих переменных
Для  $E \subset \mathbb{R}^d,\,$ отображение $f: E\rightarrow \mathbb{R}^d$ называется функцией $d$-переменных.  $$f(x) = f(x_1, ...,x_d), \;\; x \in E$$
$\text{Пусть }f:E \in \mathbb{R}^d \rightarrow R, \, x^0 \text{ предельная точка для }E. А \in \mathbb{R} \,\cup \{+\infty\} \cup \{-\infty\} - \lim f \, в \, x^0$
если $\forall\,U_{\epsilon}(A) \;\exists \,\delta > 0: f(\dot{U_{\epsilon}(x^0) \,\cap\,R}) \subset U(A)$ $$\text{При этом пишут }lim_{x \rightarrow x^0}{f(x) = A}$$  В частности, если A число, то определение с помощью неравенств имеет вид
$$\forall \, \mathcal{E} \,\exists \,\delta > 0: \, \forall x \in E: \|x - x^0\| < \delta : |f(x) - A| < \mathcal{E}$$
 Если А = $\infty$, то
 $$\forall \, \mathcal{E} \,\exists \,\delta > 0: \, \forall x \in E: \|x - x^0\| < \delta : |f(x) - A| > M$$
 $lim_{x \rightarrow x^0}{f(x) = A}$ тогда и только, когда для любой последовательности ${x^n} \subset E \;  \backslash \{x^0\}$, такой что
 $lim_{n \rightarrow \infty}{x^n = x^0}$, выполняется $lim_{n \rightarrow \infty}{f(x^n) = A}$
